# CopyAvatar

Sağ tık menüsüne avatar URL'sini kopyalama seçeneği ekleyen Vencord plugin'i.

## Özellikler

- Kullanıcıya sağ tıkla → "Avatarı Kopyala"
- Sunucu avatarı varsa onu kopyalar
- 4096px yüksek çözünürlüklü URL

## Kurulum

### UserpluginInstaller ile
```
https://github.com/VerfiredVencordPlugins/CopyAvatar
```

### Manuel
```bash
cd Vencord/src/userplugins
git clone https://github.com/VerfiredVencordPlugins/CopyAvatar.git
cd ../..
pnpm build
```
